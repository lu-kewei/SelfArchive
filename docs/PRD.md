# 产品需求文档 (PRD) 摘要 - SelfArchive

## 1. 产品概述
SelfArchive 是一款旨在帮助用户进行长期自我探索的工具，采用“侦探墙”隐喻，通过视觉化的卡片与连线，构建用户的自我画像。

## 2. 核心体验
- **隐喻**: 侦探墙 (Detective Wall) + 档案盒 (Archive Box)。
- **视觉**: 高质感拟物（毛毡、纸张、麻绳、图钉），强调真实感与沉浸感。
- **交互**: 自由拖拽、缩放、连线、归档。

## 3. 功能模块
### 3.1 侦探墙 (Board Scene)
- **画布**: 固定尺寸 (3000x1800)，无限视口，支持双指缩放/平移。
- **元素**:
  - **图腾卡 (TotemCard)**: 核心节点，代表自我象征。
  - **主题卡 (ThemeCard)**: 11个预置维度（喜欢、讨厌等），自动连接图腾。
  - **线索卡 (ClueCard)**: 用户创建，包含具体内容，通过标签自动连接主题卡。
- **操作**: 拖拽移动、缩放手柄、内容内联编辑、删除确认、主题/样式选择。

### 3.2 档案盒 (当前实现)
- **入口**: 右下角档案按钮打开 `ArchiveBox` 底部弹窗。
- **功能**: 恢复归档卡片、永久删除归档卡片。

## 4. 关键规则
- **连线逻辑**: 
  - 图腾 <-> 主题 (自动)。
  - 线索 <-> 主题 (基于标签自动匹配)。
  - 连线锚点位于图钉位置，而非卡片中心。
- **数据存储**: 本地离线存储 (Isar)，无账号体系。

## 5. 交付里程碑
- **M1**: 基础侦探墙、核心卡片逻辑、自动连线、本地存储。
- **M2**: 视觉风格包、多形状卡片与缩放交互。
- **M3**: 归档系统完善与更丰富动效。
